## Why

用户需要在桌面应用中切换浅色/深色模式，以减少长时间使用的视觉疲劳。暗色模式已成为现代应用的标准功能，提升用户体验和应用的现代感。

## What Changes

- 在 `User` 组件的菜单中增加"外观设置"入口，包含浅色模式、深色模式、跟随系统三个选项
- 实现主题切换功能，支持亮色、暗色和跟随系统三种模式
- 主题选择会持久化存储，下次启动时应用记住用户的选择
- 修改前端现有的硬编码样式，使用 CSS 变量或 Ant Design 主题变量支持模式切换
- 参考 Ant Design 官方文档实现主题切换方案

## Capabilities

### New Capabilities

- `theme-switching`: 主题切换功能，支持浅色、深色、跟随系统三种模式，包含 UI 切换入口和持久化存储

### Modified Capabilities

- 无

## Impact

- **前端组件**: User 组件增加菜单项
- **前端样式**: 需要将硬编码颜色迁移到 CSS 变量或 Ant Design 主题 token
- **配置管理**: 需要在配置服务中增加主题相关的配置项
