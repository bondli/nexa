# Nexa - AI 知识库桌面应用

## 需求功能

- 实现知识库页面和功能
  - 参考笔记本的实现，功能基本对齐，就是常规的CRUD
  - 知识库页面位置：`/frontend/pages/Knowledge/KnowBase.tsx`
  - 目前这个目录下的文件有一些TS错误，先不用管，按照笔记本页面进行对齐
  - 目前目录结构新建和准备好了
  - 具体实现：
    - 知识库对应笔记本页面（/frontend/pages/Knowledge/Note.tsx）笔记分类（Category.tsx），差异点是没有虚拟分类的存在
    - 知识库下的文档对应笔记本页面的笔记（Notes.tsx）
    - 知识库下的文档支持上传，而不是编辑器输入
    - 但知识库的文档是可以通过markdown编辑器来预览
    - 知识库页面UI上左边的知识库列表改成antd的Card组件来显示一个个知识库，点击一个知识库时候右边就加载该知识库下的文档
    - 首次进入知识库页面就加载第一个知识库下的文档
    - 每个文档在列表中有一个下载按钮，可以进行文档的下载
