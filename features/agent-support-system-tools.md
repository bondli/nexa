# Nexa - AI 知识库桌面应用

## 需求功能
- agent支持系统工具调用
- 具体功能点：
  - 在桌面端聊天页面，我和agent聊天，当我提到查看xx目录下文件的时候，能调用系统的一个命令来实现
  - 比如我说我的电脑桌面有个名叫“订单”的excel文件，帮我分析里面的订单数据，然后agent会调用一些系统的命令，比如ls/cat等读取到文件的内容，然后调用LLM去分析，最终告诉我分析的结果

### 目标
- 在桌面端聊天页面，识别用户的聊天信息，能调用相关的系统工具实现任务

### 上下文
- 文章：
  - 前端代码：./frontend/pages/ChatBox/index.tsx，这是入口文件
  - 后端接口：./server/controllers/chat-controller.ts下chatToLLM函数，里面有调用agent，agent里面有工具的实现


### 验收标准
- 进入桌面端，进入聊天页面，发送消息：电脑桌面有个名叫“订单”的excel文件，帮我分析里面的订单数据
- 然后agent会调用系统的相关工具，实现任务的执行和结果的返回
