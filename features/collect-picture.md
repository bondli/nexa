# Nexa - AI 知识库桌面应用

## 需求功能
- 在已有的浏览器插件上支持一键收藏图片，当鼠标点击网页中图片的时候，出现一个收藏的小角标，点击的时候打开一个弹层，这个弹层UI表现是：
  - 下拉框，选择图片分类（图片分类从服务端获取图片分类的数据）
  - 输入框，输入图片描述（可选）
  - 一键收藏（将图片写入图片表）
- 在frontend/pages下新增一个页面“Picture”,里面是收藏的图片，UI表现和笔记本一样，左侧是图片分类，右边是图片列表（Card组件来展示）
 - 图片分类的增删改查,和笔记本的交互形式一样
 - 图片列表按时间倒序展示，大小固定，支持删除和查看大图
- 在server下实现图片相关的API接口和数据库操作
  - 图片分类的增删改查
  - 图片的增删改查
  - 图片表必须有的字段：
    - id
    - path：存放在硬盘上的相对路径
    - name：图片名称
    - description：描述，后期用于搜索
    - categoryId
    - createdAt
    - updatedAt
- 上传图片使用已有的接口：/common/uploadImage
