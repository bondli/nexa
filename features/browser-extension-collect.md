# Nexa - AI 知识库桌面应用

## 需求功能
- 浏览器插件：一键提取当前页面的url内容成markdown格式文本，提取的内容采用markdown编辑器展示，支持将内容保存到指定的笔记分类下;
- 浏览器插件布局：“一键提取”按钮，markdown编辑器（提取到内容之后才出现），选择笔记分类选择框（提取到内容之后才出现），“保存到笔记”按钮（提取到内容之后才出现），插件界面是悬浮在页面右侧，高度随着内容增加而增加，但最大不超过1000px;
- 插件交互：点击浏览器插件图标 -> 页面出现浮层界面 -> 界面上只有“一键提取”按钮，点击“一键提取”按钮，按钮文案变成“提取中...”，提取到了内容出现markdown编辑器，内容填充到编辑器中，编辑器下方有选择笔记分类选择框，“保存到笔记”按钮；
- 点击“保存到笔记”按钮，执行服务端接口调用，进行保存，因为保存接口需要用户ID，所以插件首次使用需要检登录态，没有登录会出现登录界面，先完成登录，然后记录登录态；
- 插件的实现要求：
  - 需要用到typescript+antd的v6版本（尽量避免使用div来布局，多考虑使用antd内有的布局组件），最好采用vite来构建可安装的包，构建产物也是在dist目录下；
  - 插件代码存放目录：browser-extension，已经创建好了，内部就是源码，package.json在根目录下，构建脚本尽量和现有的构建脚本整合；
  - 服务端接口做成配置，用到的接口有：保存到笔记，拉取笔记分类，用户登录，这些接口目前在server下均已有了；
  - 页面内容的获取最傻瓜的是分析dom结构，可以考虑一些先进的实现方案，比如AIAgent，或者调用playwright这类东西，需要你给到最佳方案；
